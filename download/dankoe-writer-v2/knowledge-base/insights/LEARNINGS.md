### [LRN-20240115-001] 三句开头法

**学习日期**: 2024-01-15
**来源**: 基础模式库
**类型**: technique
**可复用程度**: ⭐⭐⭐⭐⭐
**应用次数**: 0

#### 核心发现
开头三句话决定读者是否继续阅读，需要精心设计。

#### 详细分析
Dan Koe 的文章开头通常遵循一个模式：
1. 第一句：抓住注意力（钩子）
2. 第二句：制造认知冲突（悬念）
3. 第三句：承诺价值（为什么要读下去）

#### 如何应用
写任何文章时，单独审视前三句话：
- 第一句是否有足够的吸引力？
- 第二句是否制造了悬念或冲突？
- 第三句是否给了读者继续阅读的理由？

#### 应用示例
```
原文：
"大多数人每天都很忙。"（太平淡）

优化：
"大多数人每天都很忙，但很少有人真正高效。"（加入对比）
"如果你也是其中之一，这篇文章可能会改变你的工作方式。"（承诺价值）
```

### [LRN-20240115-002] 转折词的力量

**学习日期**: 2024-01-15
**来源**: 基础模式库
**类型**: technique
**可复用程度**: ⭐⭐⭐⭐
**应用次数**: 0

#### 核心发现
使用转折词可以制造阅读节奏和认知冲击。

#### 详细分析
转折词的使用规律：
- "但是" 之后通常是核心观点
- "其实" 之后通常是颠覆认知的真相
- "所以" 之后通常是行动指引

#### 如何应用
1. 在关键观点前使用转折词
2. 利用转折词调整文章节奏
3. 转折词后的内容要值得期待

#### 应用示例
```
普通表达：
"努力工作很重要，但方向更重要。"

Dan Koe 风格：
"努力工作？当然重要。但其实，方向比努力重要100倍。"
```
