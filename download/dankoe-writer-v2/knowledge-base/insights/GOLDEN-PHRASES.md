## 金句创作公式

### 公式 1：对比法

```
"大多数人追求[常量]，高手却在做[变量]。"
```

**示例**:
- "大多数人追求更多选项，高手却在做减法。"
- "大多数人追求稳定，高手却在拥抱不确定。"

### 公式 2：否定法

```
"[常见观点]？错。[真实原因]。"
```

**示例**:
- "努力就能成功？错。方向比努力重要100倍。"
- "时间越多越好？错。限制反而激发创造力。"

### 公式 3：重新定义

```
"[概念]不是[A]，而是[B]。"
```

**示例**:
- "自由不是想做什么就做什么，而是不想做什么就可以不做什么。"
- "学习不是输入，而是重构。"

### 公式 4：递进法

```
"不仅...更要..."
```

**示例**:
- "不仅要学会努力，更要学会不努力。"
- "不仅要读万卷书，更要行万里路。"

### 公式 5：本质揭示

```
"[事物]的本质，不是[表象]，而是[内核]。"
```

**示例**:
- "学习的本质，不是记忆，而是改变。"
- "赚钱的本质，不是努力，而是价值交换。"
