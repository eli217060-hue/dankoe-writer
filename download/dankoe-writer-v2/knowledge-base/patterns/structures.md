# 文章结构模式库

收集各种有效的文章结构模式，用于组织内容逻辑。

---

## 已验证结构模式

### 1. PASTOR 结构

**适用场景**: 深度长文、销售文案、知识分享

```
P - Problem（问题）
  └─ 开篇点出读者面临的具体痛点
     └─ 用提问或场景描述引发共鸣

A - Amplify（放大）
  └─ 放大问题的后果和情绪
     └─ "如果不解决，会怎样..."

S - Story/Solution（故事/方案）
  └─ 分享经历或给出解决方案框架
     └─ 可以是个人故事 + 方法论

T - Testimony/Transformation（证据/转变）
  └─ 具体案例或数据支撑
     └─ "我用这个方法后..."或"我的学员..."

O - Offer（提供）
  └─ 给出具体的行动框架或步骤
     └─ 分步骤、可执行

R - Response（回应）
  └─ 明确的下一步行动号召
     └─ "今天就试试..."
```

**字数分配建议**:
| 部分 | 占比 | 字数(1500字文) |
|------|------|----------------|
| P | 10% | 150字 |
| A | 15% | 225字 |
| S | 30% | 450字 |
| T | 20% | 300字 |
| O | 15% | 225字 |
| R | 10% | 150字 |

### 2. AIDA 结构

**适用场景**: 短内容、推文、短视频脚本

```
A - Attention（注意）
  └─ 用一句震惊的话抓住眼球
     └─ 1-2句话，不超过30字

I - Interest（兴趣）
  └─ 用一个反直觉观点引发好奇
     └─ "真正的原因是..."

D - Desire（欲望）
  └─ 展示转变后的美好图景
     └─ "想象一下..."

A - Action（行动）
  └─ 给出一个简单可执行的第一步
     └─ "今晚就..."
```

**字数分配建议**:
| 部分 | 占比 | 字数(500字文) |
|------|------|---------------|
| A | 10% | 50字 |
| I | 25% | 125字 |
| D | 35% | 175字 |
| A | 30% | 150字 |

### 3. PAS 结构

**适用场景**: 销售文案、解决问题型内容

```
P - Problem（问题）
  └─ 描述具体、可感知的痛点
     └─ 越具体越好

A - Agitate（煽动）
  └─ 为什么这个问题一直存在？
  └─ 不解决的代价是什么？
     └─ 情感层面 + 理性层面

S - Solution（解决）
  └─ 你的独特解决方案
  └─ 具体步骤
     └─ 可立即执行
```

---

### 4. 3问深挖结构

**适用场景**: 深度分析、认知类内容

```
第一层：WHAT（是什么？）
  ├─ 这件事的本质是什么？
  ├─ 大多数人误解了什么？
  └─ 真正的定义是什么？

第二层：HOW（怎么做？）
  ├─ 具体的执行步骤是什么？
  ├─ 有什么隐藏的关键细节？
  └─ 常见错误有哪些？

第三层：WHY（为什么？）
  ├─ 这件事为什么重要？
  ├─ 底层逻辑是什么？
  └─ 与其他事物有什么关联？
```

---

### 5. 反共识结构

**适用场景**: 挑战常规认知、爆款内容

```
开头：提出一个反直觉的观点
  └─ "大多数人认为X是对的，但其实..."
  └─ 制造认知冲突

主体：三个支撑论据
  ├─ 论据1：为什么常规方法会失败
  │   └─ 逻辑 + 案例
  ├─ 论据2：真实案例或数据
  │   └─ 具体、可信
  └─ 论据3：更好的替代方案
      └─ 给出解决方案

结尾：具体行动建议
  └─ "从今天开始，试着..."
```

## 结构变体

### 变体 1：故事先行型 PASTOR

```
S - Story 先讲一个故事
P - 从故事中提炼问题
A - 放大问题的普遍性
S - 给出解决方案
T - 证据支撑
O - 具体步骤
R - 行动号召
```

**适用**: 个人经历类内容

---

### 变体 2：数据开场型

```
数据震撼 → 问题提出 → 原因分析 → 解决方案 → 行动号召
```

**适用**: 专业分析类内容

## 结构选择决策树

```
你的内容是？
├─ 解决具体问题
│   └─ 使用 PAS 或 PASTOR
│
├─ 颠覆常规认知
│   └─ 使用反共识结构
│
├─ 深度分析主题
│   └─ 使用 3问深挖法
│
├─ 短内容/社交媒体
│   └─ 使用 AIDA
│
└─ 个人故事分享
    └─ 使用 故事先行型 PASTOR
```
