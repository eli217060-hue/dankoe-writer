# 开头钩子模式库

收集各种有效的开头钩子模式，用于抓住读者注意力。

---

## 钩子模式分类

### 1. 痛点触发型

**模式公式**:
```
[读者正在经历的痛苦描述]？
[放大痛苦的后果]？
```

**示例**:
- "你是不是每天忙得要死，却感觉一事无成？"
- "你是不是定了无数目标，却一个都没完成？"

**适用场景**:
- 解决问题型文章
- 方法论内容
- 自我提升类

---

### 2. 反直觉型

**模式公式**:
```
[普遍认知]？错。
[反直觉的真相]。
```

**示例**:
- "努力就能成功？错。方向比努力重要100倍。"
- "99%的人都在用错误的方式努力。"

**适用场景**:
- 颠覆认知型文章
- 建立专业形象
- 引发讨论

---

### 3. 数据震撼型

**模式公式**:
```
[令人震惊的数据/事实]。
这意味着[深刻的含义]。
```

**示例**:
- "我研究了100个成功人士，发现他们有一个共同点：都不努力工作。"
- "过去5年，我帮助了超过10000人实现副业变现。"

**适用场景**:
- 建立权威
- 案例分析
- 经验分享

---

### 4. 故事引入型

**模式公式**:
```
[时间/场景设置]。
[意想不到的事件]。
[由此引出的主题]。
```

**示例**:
- "三年前，我还是一个朝九晚五的上班族。今天，我边旅行边赚钱。"
- "上个月，我做了一个决定，彻底改变了我的人生轨迹。"

**适用场景**:
- 个人故事
- 转变经历
- 情感共鸣

---

### 5. 提问引导型

**模式公式**:
```
你有没有想过：[深刻的问题]？
大多数人认为是[A]，但其实是[B]。
```

**示例**:
- "你有没有想过：为什么有些人看起来毫不费力就能成功？"
- "你有没有发现：越努力的人，往往越不成功？"

**适用场景**:
- 深度分析
- 启发式内容
- 思考引导

---

## 钩子选择指南

| 文章类型 | 推荐钩子 | 原因 |
|----------|----------|------|
| 方法论 | 痛点触发 | 直接建立共鸣 |
| 认知颠覆 | 反直觉 | 制造认知冲突 |
| 经验分享 | 数据震撼 | 建立可信度 |
| 个人故事 | 故事引入 | 自然过渡 |
| 深度思考 | 提问引导 | 激发好奇心 |
