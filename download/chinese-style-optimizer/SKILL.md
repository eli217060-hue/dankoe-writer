# 中文文笔优化器

> 基于咪蒙等优质中文范例优化文章文笔
>
> **⚠️ 核心原则：只优化文笔和表达方式，绝不改变原有框架和结构**

---

## 一、定位与分工

### 与 Dankoe Writer 的分工

```
Dankoe Writer 负责（不动）：
├─ 结构设计 ← 不动
├─ 框架选择（PASTOR/AIDA等）← 不动
├─ 逻辑安排 ← 不动
├─ 节奏控制 ← 不动
└─ 观点输出 ← 不动

本模块负责（只改这些）：
├─ 句式优化：生硬→自然
├─ 口语化：书面→口语
├─ 情绪表达：平淡→有情绪
├─ 画面感：抽象→具体
└─ 金句打磨：提炼可传播的句子
```

### 工作流程

```
Dankoe Writer 生成初稿（结构+框架已定）
           ↓
Chinese Style Optimizer 优化文笔
           ↓
    只改表达方式，不动结构
```

## 三、文笔优化技巧

### 3.1 去翻译腔

| 生硬表达 | 自然表达 |
|---------|---------|
| 因此 | 所以嘛 |
| 然而 | 但是呢 |
| 综上所述 | 说白了就是 |
| 由此可见 | 你看 |
| 值得注意的是 | 有个事儿你得知道 |

### 3.2 去书面语

| 书面语 | 口语化 |
|--------|--------|
| 进行了一次思考 | 想了想 |
| 给予了高度评价 | 说特别棒 |
| 呈现出一种趋势 | 变得越来越... |
| 具有十分重要的意义 | 特别重要 |

### 3.3 情绪表达

| 平淡 | 有情绪 |
|------|--------|
| 这让人很生气 | 我气得把手机都摔了 |
| 这件事很遗憾 | 这件事让我难过了整整三年 |
| 我很无语 | 我：？？？？ |

### 3.4 画面感

| 抽象表达 | 画面表达 |
|---------|---------|
| 她很焦虑 | 凌晨三点，她盯着天花板 |
| 他很孤独 | 他一个人吃火锅，对面放着包 |
| 生活很苦 | 住在地下室，衣服总有股霉味儿 |

### 3.5 金句公式

**对比型**：A不是X，而是Y
```
大多数婚姻不是死于无性，而是死于无话
```

**否定型**：X？错。Y。
```
努力就能成功？错。方向比努力重要100倍
```

**重定义型**：X不是Y，而是Z
```
睡眠不是任务，是礼物
```

## 八、核心检查清单

优化完成后检查：

**只改了文笔**：
- [ ] 句式是否更自然？
- [ ] 是否去掉了翻译腔？
- [ ] 情绪表达是否更到位？
- [ ] 画面感是否更强？
- [ ] 金句是否更出彩？

**没动结构**：
- [ ] 段落数量一致？
- [ ] 框架位置不变？
- [ ] 论证顺序不变？
- [ ] 核心观点不变？
- [ ] 篇幅相当？

---

*记住：结构是骨架，文笔是血肉。只优化血肉，不动骨架。*
