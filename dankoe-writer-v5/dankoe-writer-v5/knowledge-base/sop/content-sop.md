## SOP-001：反共识型

### 适用场景
- 挑战行业主流观点
- 提供颠覆性视角
- 建立思想领导力

### 结构模板

```
[钩子：反直觉陈述，参见 patterns/hooks.md 模式 4]

一、[X] 曾经有效
  [历史背景，为什么过去成功]

二、为什么现在失效了
  [时代变化分析，核心矛盾揭示]

三、新的成功路径
  [你的反论，具体策略]

四、行动清单
  [3-5 个具体步骤]

[结尾金句，参见 patterns/golden-phrases.md]
```

### 写作要点
1. 开场否定主流，但先承认它曾经有道理
2. 诊断问题根源要深入到心理和社会层面，不停留在表面
3. 新范式必须同样简单易懂，但更有解释力
4. 不能只批判不建设，必须给可执行的步骤

## SOP-002：方法论型

### 适用场景
- 分享具体技巧
- 教授可执行方法
- 构建个人系统

### 结构模板

```
[钩子：问题诊断或个人故事，参见 patterns/hooks.md 模式 3 或 5]

一、为什么大多数人失败
  [指出常见误区，建立共鸣]

二、核心框架
  [你的方法论，用类比让它易懂]

三、具体步骤
  Step 1: [...]
  Step 2: [...]
  Step 3: [...]

四、为什么这样有效
  [回到底层逻辑，解释机制]

[行动号召]
```

### 写作要点
1. 先说"为什么你的旧方法不行"再给新方法
2. 步骤必须具体到读者读完就能做
3. 解释"为什么有效"让方法论有深度，不只是清单

- 06-future-of-productivity.md

## SOP-003：规则列表型

### 适用场景
- 整合多年经验
- 创建参考清单
- 系统化个人哲学

### 结构模板

```
[钩子：数据冲击或权威引用，参见 patterns/hooks.md 模式 1 或 2]

Rule 1: [核心主张]
  [2-3 段解释 + 示例]

Rule 2: [核心主张]
  [2-3 段解释 + 示例]

...

[结尾：升华到更高层面]
```

### 写作要点
1. 规则之间有递进关系，不是并列罗列
2. 前面的规则是后面的基础
3. 最后的规则是升华，比前面的都高一个层次

## SOP-006：对比表格

### 适用场景
- 概念对比（旧思维 vs 新思维）
- 方法对比（传统 vs 创新）
- 阶段对比（以前 vs 现在）

### 模板

```
| [旧概念] | [新概念] |
|----------|----------|
| [特征 1] | [特征 1] |
| [特征 2] | [特征 2] |
| [特征 3] | [特征 3] |
```

### 写作要点
- 对比要产生认知落差，不是平铺直叙
- 左列（旧）读者会点头同意，右列（新）让他们停下来思考
- 短文中避免用表格格式，用文字对比代替

## SOP-007：行动号召

### 模式 A：具体步骤型
```
今天就可以做的 3 件事：
1. [最简单的第一步]
2. [进阶步骤]
3. [深度练习]
```

### 模式 B：身份转变型
```
从今天开始，不要再问"我该怎么做"。
问自己："我要成为什么样的人？"
然后让行为自然发生。
```

### 模式 C：开放问题型（适合短文）
```
问题是——你愿意花多长时间停在原地？
```

## SOP-008：引用与权威

### 模式 A：权威引用 + 个人解读
```
> [名人/研究者的话]
这句话大多数人只看到了表面。
真正的意思是 [你的深层解读]。
```

### 模式 B：数据引用 + 洞察
```
[研究机构] 发现 [数据]。
但没人注意到的是 [你的独特洞察]。
```

### 写作要点
- 权威只是钩子，快速转向自己的框架
- 数据必须有来源（联网研究获取）
- 不要堆砌引用，1-2 个就够

## SOP-010：内容积木系统

### V5 变化说明

V5 中积木系统已从"全量生成 32 块"改为"定向填充"模式：
- 不再固定 8 类 32 块
- 根据选定大纲的需要，定向生成所需积木
- 每块积木必须标注"类型 + 情感功能"
- 每块积木必须通过质量检查（画面感/12岁/截图测试）

详见 SKILL.md Step 6 和 Step 7。

### 积木情感功能分类（V5 新增）

| 情感功能 | 作用 | 常见积木类型 |
|----------|------|-------------|
| 颠覆认知 | 让读者停下来重新思考 | 核心洞察、反常识钩子 |
| 制造共鸣 | 让读者觉得"他在说我" | 故事、问题 |
| 秒懂 | 降低理解门槛 | 类比 |
| 建立信任 | 用数据/权威增加可信度 | 数据 |
| 让人截图 | 可传播的金句 | 金句 |
| 制造紧迫 | 让读者觉得"我必须改变" | 反常识钩子、数据 |
| 提供希望 | 让读者觉得"我也能做到" | 行动步骤、故事 |
