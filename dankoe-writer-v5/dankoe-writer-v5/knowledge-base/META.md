# Dan Koe Writer Knowledge Base

## 知识库概述

本知识库基于 Dan Koe 的文章、视频和播客内容创建，配合联网研究系统，用于生成有深度、有真实数据支撑的内容。

## V5 更新说明

- **核心变化**：大纲前置，积木服务于大纲（不再是"选角度→生成32块积木→选方案"）
- **新增**：5 个完整大纲模板（短文/长文分别有不同模板）
- **新增**：三种情感原型（耐心观察者/戏剧性预言家/静默颠覆者）
- **新增**：APAG 长文框架（Attention → Perspective → Advantage → Gamify）
- **新增**：定向积木生成 + 情感功能标签
- **新增**：积木质量检查（画面感/12岁/截图/真实性测试）
- **新增**："攻击敌人"开场法
- **新增**：策略性模糊 + "保留"规则
- **新增**：节奏示范（三种情感原型各有节奏模板）
- **新增**：深度追问新增"读者说不出的感受"等 3 个问题
- **调整**：SOP 降级为参考资料，不再是组装方案的主要分类维度
- **调整**：用户决策从 3 次减少到 2 次
