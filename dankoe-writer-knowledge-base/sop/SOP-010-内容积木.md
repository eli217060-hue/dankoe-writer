# SOP-010: 内容积木系统

> 内容积木库 = 预制内容单元，按需组装成文

---

## 一、积木库架构

### 积木库结构

```
内容积木库
├── 核心洞察    # 深度观点，文章的灵魂
├── 金句        # 短小精炼，传播利器
├── 故事        # 场景代入，证据支撑
├── 数据        # 科学背书，可信度加成
├── 类比        # 形象比喻，降低理解门槛
├── 反常识钩子  # 开场引爆，打破认知
├── 行动步骤    # 具体方法，立即可做
└── 问题        # 互动引导，评论区杀手
```

## 二、积木类别详解

### 1. 核心洞察

**定义**：文章的灵魂观点，有深度、有洞察、能改变认知

**写作要点**：
- 不是"精油能助眠"，而是"精油是关机键"
- 要触及本质，不是停留在表面
- 一个洞察 = 一个认知转变

**示例**：
```
#1：精油不是安眠药，它是"关机键"。失眠的本质是大脑卡在白天模式，精油通过嗅觉神经直达边缘系统，帮大脑完成从兴奋到放松的模式切换。

#2：嗅觉是唯一一个不经过"理性大脑"就能直达情绪中枢的感官。这就是为什么闻到一个味道你会先放松、后才意识到自己放松了——精油绕过了你那个"停不下来的脑子"。

#3：失眠的人最大的误区是"努力睡觉"。越努力越清醒，因为"努力"本身就是一种大脑激活状态。精油的价值不是让你"更努力入睡"，而是让你"不需要努力"。
```

---

### 2. 金句

**定义**：短小精炼，好记好传播，适合做标题或结尾

**写作要点**：
- 一句话说透一件事
- 有对比、有画面、有情绪
- 让人想截图转发

**示例**：
```
#1：你的失眠不是因为不够累，是因为大脑不知道怎么停下来。
#2：精油不治失眠，它治的是"你躺下了但大脑还在开会"。
#3：最好的助眠，不是让你昏过去，是让你的身体想起"原来可以松下来"。
#4：安眠药是强制关机，精油是让电脑自己进入休眠。
#5：你缺的不是一瓶精油，你缺的是一个"从白天切换到夜晚"的仪式。
```

---

### 3. 故事

**定义**：场景代入或证据支撑，让读者"看见"

**类型标签**：
- `[可创作场景]`：虚构但真实的代入场景
- `[来自研究]`：真实的实验或案例
- `[个人经历]`：自己的故事

**示例**：
```
#1 [可创作场景]：凌晨一点，你躺在床上，身体已经累得不行，但脑子里还在复盘今天的会议、明天的 deadline、上周没回的微信……你翻了个身，看了眼手机，01:17。你叹了口气，心想"我明明这么累，为什么就是睡不着？"

#2 [来自研究]：日本鹿儿岛大学2018年实验——把焦虑的小鼠暴露在芳樟醇蒸气下，焦虑程度大幅下降。但如果破坏小鼠的嗅觉神经元，芳樟醇完全失效。说明这个成分的镇静作用是通过"闻"实现的，不是通过血液。

#3 [来自研究]：上海一位消费者买了薰衣草精油想助眠，用了之后反而更兴奋。后来发现那瓶精油掺了合成香精——掺假的精油不但不助眠，可能适得其反。
```

---

### 4. 数据

**定义**：科学背书，让观点有据可查

**写作要点**：
- 来源可信（研究机构、学术期刊）
- 数据具体（数字、浓度、条件）
- 可引用、可查证

**示例**：
```
#1：吸入薰衣草精油后，大脑产生更多θ波和α波（放松状态的脑电波），这是2016年研究实测的结果。
#2：国内研究证实1%浓度的薰衣草精油对原发性失眠有效——关键词：1%浓度、原发性失眠。不是随便一瓶、不是所有失眠都管用。
#3：《中国睡眠研究报告2024》数据，国人平均睡眠指数仅62.61分，刚过及格线。
#4：芳香按摩比单纯嗅吸效果更明显（56人随机对照实验），因为按摩手法帮助芳香分子通过皮肤进入血液循环。
```

---

### 5. 类比

**定义**：形象比喻，把抽象变具体，降低理解门槛

**写作要点**：
- 用日常熟悉的事物类比
- 一句话说清楚复杂概念
- 让人"秒懂"

**示例**：
```
#1：精油是"人造日落"。自然界里动物靠日落切换到休息模式，但现代人的手机屏幕把日落取消了。精油的香气就是你为自己制造的那个"天黑了"的信号。
#2：安眠药是强制关机（长按电源键），精油是让系统自己进入休眠模式——进程一个个关闭，优雅退出。
#3：你的大脑像一间灯火通明的办公室，失眠就是到了下班时间但灯全亮着、空调还开着。精油不是去拉闸断电，是一盏一盏帮你关灯。
#4：嗅觉是大脑的"后门"。视觉、听觉的信号都要经过理性中枢"审核"，但气味直接走后门，绕过你那个停不下来的思考系统。
```

---

### 6. 反常识钩子

**定义**：开场引爆，打破认知，制造"我想点进去看看"

**写作要点**：
- 打破读者预期
- 制造认知冲突
- 但要真诚，不是标题党

**示例**：
```
#1：做了X年芳疗师，我从不跟客户说"这个精油能治失眠"。
#2：用了精油还是睡不着？不是精油没用，是你用的时机全错了。
#3：失眠最大的敌人不是压力，是你"努力想睡着"这件事本身。
#4：薰衣草是最有名的助眠精油——但你闻到的那瓶，可能根本不是真的。
#5：我建议所有失眠的客户：别急着"治失眠"，先学会"关机"。
```

---

### 7. 行动步骤

**定义**：具体方法，读者今晚就能做

**写作要点**：
- 越具体越好（时间、用量、方法）
- 有条件限定（什么情况用什么方法）
- 可替代方案（如果A不行，试试B）

**示例**：
```
#1 [30分钟仪式]：睡前30分钟开香薰机，滴3滴真正薰衣草（Lavandula angustifolia）。不是躺到床上才开始——给大脑一段"天黑了"的过渡时间。

#2 [辨真假]：买精油看瓶身有没有标拉丁学名。写了"薰衣草香氛"但没有学名的，大概率是合成香精。

#3 [按摩加持]：如果只是嗅吸效果一般，试试在手腕内侧和太阳穴各涂一滴（用基础油稀释到1%），按摩帮助芳香分子经皮肤吸收，研究证实效果更好。

#4 [替代方案]：薰衣草对你无感？试试佛手柑+罗马洋甘菊，适合焦虑型失眠；如果是脑子停不下来的"思虑型"失眠，试试岩兰草。

#5 [配合动作]：扩香的同时做3次深呼吸——鼻吸4秒、屏息4秒、口呼6秒。呼吸节奏本身也在告诉大脑"该切换了"。
```

---

### 8. 问题

**定义**：互动引导，激发评论和转发

**写作要点**：
- 问到心坎里
- 让人想回答
- 有时是反思性问题

**示例**：
```
#1：你有没有想过，失眠的问题可能根本不在"睡觉"上？
#2：每天晚上，你有没有一个固定的"告诉大脑该休息了"的信号？
#3：你买的精油瓶子上，有写拉丁学名吗？
```

## 三、组装方案设计

### 组装原则

1. **方案要有明显差异**：不同调性、不同受众、不同情绪流动
2. **每方案标注逻辑和调性**：让用户知道选的是什么
3. **积木编号清晰**：方便用户混搭

### 组装模板（基于 Dan Koe 结构）

#### 方案 1：专业权威型
```
反常识钩子 → 核心洞察 → 类比 → 行动步骤 → 金句
```
- 逻辑：身份开场 → 重塑认知 → 形象比喻 → 今晚就能做 → 金句收尾
- 调性：专业、清醒、有距离感的可信度
- 适用：建立专家形象、知识分享

#### 方案 2：共情故事型
```
故事 → 核心洞察 → 类比 → 行动步骤 → 金句
```
- 逻辑：场景代入 → 揭示真相 → 形象比喻 → 具体方法 → 温暖收尾
- 调性：温柔、懂你、像朋友在跟你说话
- 适用：情感共鸣、生活方式分享

#### 方案 3：实用避坑型
```
反常识钩子 → 故事（翻车案例） → 数据 → 行动步骤 → 问题
```
- 逻辑：打破认知 → 真实案例 → 数据支撑 → 解决方案 → 互动引导
- 调性：犀利、干货、有"种草拔草"的小红书感
- 适用：产品推荐、避坑指南

     ↓
