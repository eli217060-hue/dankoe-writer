## 核心观点

### 1. "价值"的通货膨胀

过去:
- 提供有价值的信息 = 成功
- 好内容会自然传播
- 创作者稀缺

现在:
- AI 可以生成无限"有价值"的内容
- 注意力是稀缺资源
- 每个人都在生产内容

### 2. 新的内容范式

**从价值型到视角型**:

| 价值型内容 | 视角型内容 |
|------------|------------|
| "5个提高效率的方法" | "为什么你不需要效率" |
| 可被 AI 复制 | 独特世界观 |
| 信息传递 | 思维框架 |
| 教你做什么 | 改变你怎么看 |

### 3. 如何脱颖而出

在"垃圾海"中，你需要：
1. **独特视角** - 别人无法复制的世界观
2. **深度洞察** - 超越表面的思考
3. **个人经验** - 真实的故事和经历
4. **一致性** - 长期构建的品牌认知

## Dan Koe 的写作技巧

### 1. 否定主流观点

```markdown
Everyone says [X].
But they're wrong.
Here's why...
```

### 2. 重新定义术语

```
"Value" isn't what you think it is.
True value is [重新定义].
```

### 3. 对比论证

```
[旧方法]
This used to work because...

[为什么不再有效]
But now AI can...

[新方法]
The new paradigm is...
```
