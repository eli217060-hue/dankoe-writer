> "The skill everyone tells you is unethical"

## 为什么这篇文章有效

### 1. 标题策略

**好奇心缺口 + 身份挑战**：
- "Every ambitious person" = 读者自我选择（我是有野心的人吗？）
- "must learn" = 紧迫性
- 副标题 "The skill everyone tells you is unethical" = 禁忌感，让人非看不可
- 迟迟不揭示是什么技能 → 好奇心驱动

### 2. 个人故事开场

Dan Koe 自己的创业失败史：
- 2014 年花 $999 买了个"骗子课程"
- 尝试过 dropservicing、SEO、dropshipping
- 核心问题：从来没有给自己的工作做过广告
- 最终不得不找工作

**效果**：共鸣 + 权威（"连我也犯过这个错"）

## 核心观点拆解

### 1. 广告是主权个体的技能

> "如果你不展示你的工作如何惠及他人，你就必须去做别人的工作。"

Dan Koe 的核心论点：
- 米开朗基罗给美第奇家族做广告
- 莎士比亚为商业剧院写作
- "饥饿艺术家"是现代神话
- 孔雀开屏本质上就是广告
- 广告 = 资本主义社会的自然选择

### 2. 提升感知价值的三要素

| 要素 | 说明 |
|------|------|
| 大问题 (Big Problem) | 你解决什么问题？ |
| 期望结果 (Desired Outcome) | 你承诺什么结果？ |
| 独特机制 (Unique Mechanism) | 你的独特方法是什么？ |

### 3. AI 加速学习法

具体 prompt 示范：
```
找到一篇已经表现很好的 newsletter
粘贴到 AI 对话中
让 AI 拆解：宏观结构、每段结构、心理策略、核心思想类型
→ 瞬间获得一门"迷你课程"
```

然后让 AI 引导你写自己的版本。

### 4. 学习 = 实践

> "你读多少书不重要。你是否得到了现实世界的反馈才重要。"

- 看 Photoshop 教程时必须同时开着项目
- 写作就是笔记，只是多了一点意图
- 教别人是学习最快的方式（学徒效应）

## 写作技巧分析

### 1. 结构：个人故事 → 重构认知 → 实用方法

```
开场：个人失败故事
转折：发现核心问题是"没做广告"
重构：广告不是操纵，是自然法则
方法1：三要素提升感知价值
方法2：AI 加速学习
方法3：边学边做
```

### 2. 日常类比消除抵触

"你说你讨厌营销，但你每天都在做"：
- 你描述周末时选择性呈现自己
- 你说服朋友去某个餐厅
- 你在面试中推销自己
- 你给孩子推销吃蔬菜

### 3. 不端着的"嬉皮→商人"叙事

Dan Koe 承认自己来自"spiritual hippy background"：
- 天然对广告有抵触
- 但发现不学广告就得给别人打工
- 这个叙事让同样抵触的读者放下防御

## 可复用的模式

### "你已经在做了"重构
```markdown
你说你讨厌 [争议事物]。
但你每天都在做。
你只是无意识地在做，
这让你更容易 [负面结果]。
```

### "三要素"自我营销框架
```markdown
1. 你解决的大问题是什么？
2. 你承诺的结果是什么？
3. 你的独特方法是什么？

搞定一个，你就超过大多数人。
搞定三个，赚钱不再是问题。
```

### AI 拆解学习法
```markdown
1. 找到已成功的范例
2. 让 AI 做完整拆解（结构、策略、心理）
3. 让 AI 引导你写自己的版本
4. 在真实反馈中迭代
```
