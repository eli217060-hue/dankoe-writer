# 金句公式

> 从 Dan Koe 文章中提取的金句模式，按类型分类
> 每种公式同时提供英文原版逻辑和**中文原生示例**
> 生成金句时优先使用中文原生节奏，参考 `chinese-native.md` 的速查表

---

## 一、身份重构型

**逻辑公式**：你不是 [表面问题]，你是 [深层原因]

**英文逻辑参考**：
```
You aren't where you want to be
because you aren't the person who would be there.
```

**中文原生示例**：
- "你哪是懒，你是怕做了也没用。"
- "不是路太远，是你还没变成走得到的那个人。"
- "你以为你在拖延，其实你在自我保护。"
- "不是你不够好，是你一直在用'不够好'当借口。"

**用法**：把行为问题重新定义为身份问题，让读者从"我该怎么做"转向"我该成为谁"。

---

## 二、目标与感知型

**逻辑公式**：[结果] 不取决于 [表面因素]，取决于 [深层因素]

**英文逻辑参考**：
```
A goal is a projection into the future
that acts as a lens of perception.
```

**中文原生示例**：
- "目标不是终点，是滤镜——你盯着什么，就只看得见什么。"
- "你不是没有方向，你是同时盯着太多方向。"
- "感兴趣的事不用人催，不感兴趣的事闹钟也没用。"
- "你以为你在选择，其实是你的恐惧在替你选。"

**用法**：揭示目标的隐藏功能——不是终点，而是过滤器。

---

## 三、对比型

**逻辑公式**：A 不是 X，而是 Y

**英文逻辑参考**：
```
AI doesn't replace humans. AI replaces tasks.
```

**中文原生示例**：
- "忙，不叫努力。忙对了，才叫。"
- "忙是最体面的偷懒。"
- "看似自律，实则自虐。"
- "你不是缺少自律，你是缺少安全感。"
- "收藏不是学习，执行才是。"
- "能被复制的，迟早被替代。"

**用法**：用简洁的对比制造认知落差。最强版本是前半句读者会点头同意，后半句让他们停下来思考。

**中文技巧**：
- 四字格对仗是最强的中文对比形式
- 不需要"而是""但是"，对仗结构本身就包含转折
- 越短越有力

## 四、否定型

**逻辑公式**：别再 X 了 / X？错。Y。

**英文逻辑参考**：
```
Stop trying to find your passion. Build it.
```

**中文原生示例**：
- "别再'找'热情了。热情是做出来的，不是找到的。"
- "方向比努力重要一百倍。"
- "别再每天列清单了。你的问题不是没有计划，是计划太多。"
- "别学什么时间管理了。你缺的不是方法，是取舍的勇气。"

**用法**：直接否定常见行为或信念，制造紧张感后给出替代方案。

**中文技巧**：
- 否定后面的替代方案用短句，不要展开论证
- "X？错。Y。" 这个节奏在中文里很有力
- 否定的对象越具体越好——"别再列清单"比"别再做无效的事"有力十倍

## 五、重定义型

**逻辑公式**：什么是 X？X 不是 Y，而是 Z

**英文逻辑参考**：
```
Intelligence is the ability to iterate and persist.
Productivity isn't doing more. It's doing what matters.
```

**中文原生示例**：
- "什么叫专注？不是死磕一件事，是敢于放弃其他九件。"
- "创作不是输出内容，是构建世界观。"
- "所谓成长，就是把'不敢'变成'试试'的过程。"
- "自由不是想做什么就做什么，是可以不做什么。"

**用法**：重新定义一个常见概念，给读者一个新的思考框架。

**中文技巧**：
- 重定义的关键是把抽象概念拉到日常行为层面
- "什么叫X？" 开头 + 一句话重定义 = 完整的短帖素材

---

## 六、行为洞察型

**逻辑公式**：你之所以 [负面结果]，是因为 [隐藏原因]

**英文逻辑参考**：
```
You aren't where you want to be
because you aren't the person who would be there.
```

**中文原生示例**：
- "拖延不是因为懒。拖延是因为你在保护自己免受评判。"
- "你之所以总是半途而废，是因为你从来没有真正开始过——你只是在'准备开始'。"
- "你不是不知道该怎么做。你是知道了但不想承认做起来没那么容易。"
- "你总说'等我准备好了就开始'。可你等的不是准备好，是确定不会失败。"

**用法**：揭示行为背后的心理机制，让读者产生"被看穿"的感觉。

**中文技巧**：
- 用"你"直接对话，不用"人们""大众"
- 隐藏原因应该是读者隐约感觉到但不敢说出口的东西
- 说完洞察后不要立刻给解法，让读者在"被看穿"的感觉里停留一会

## 金句使用指南

| 位置 | 适合的金句类型 |
|------|----------------|
| 开头钩子 | 否定型、对比型（制造冲击） |
| 段落收尾 | 重定义型、身份重构型（升华观点） |
| 文章结尾 | 行为洞察型（留下深刻印象） |
| 独立短帖 | 任何类型（金句本身就是完整内容） |

| 语气 | 适合的金句类型 |
|------|----------------|
| 犀利 | 否定型、对比型 |
| 温暖 | 身份重构型、行为洞察型 |
| 哲学 | 重定义型、目标与感知型 |

**中文金句自检清单**：
```
□ 念出来是否顺口？（不顺口 → 改句式）
□ 是否像翻译过来的？（是 → 参考 chinese-native.md 重写）
□ 删掉"而是""但是"后是否仍然成立？（不成立 → 对比不够强）
□ 读者看到会不会想截图？（不会 → 还不够好）
```
