> 确保内容"生在中文里"而不是"从英文翻译过来"。
>
> 生成金句、钩子、故事、标题时参考本文件。

## 一、中文节奏工具箱

### 1.1 四字格对仗

中文最强的节奏武器。两个四字格对仗，胜过一段论证。

```
用法：核心观点浓缩成两组四字格，形成对比或递进

示例：
- "看似自律，实则自虐"
- "越想入睡，越难入睡"
- "方向不对，努力白费"
- "不是路远，是你没走"
- "嘴上躺平，心里不甘"
- "表面合群，内心荒芜"

技巧：
- 前四字 = 表象/大众认知
- 后四字 = 真相/反转
- 两组之间有转折或递进关系
- 不需要"然而""但是"，对仗本身就是转折
```

### 1.2 三段递进

三个短句，层层加重。中文演讲和金句的经典结构。

```
示例：
- "你不是懒。你不是笨。你是怕。"
- "第一年，怀疑行业。第二年，怀疑公司。第三年，怀疑自己。"
- "删掉了。又写回来。又删掉了。"
- "不是不想改变。不是不能改变。是改变的代价，你还没准备好付。"

技巧：
- 前两句建立节奏预期
- 第三句打破预期或升华
- 每句尽量等长（视觉节奏感）
- 第三句可以突然变长或变短，制造张力
```

### 1.3 反问收尾

中文的反问比陈述有力十倍。结尾用反问，读者会把答案留在心里。

```
示例：
- "可是，谁规定人生只能有一条路？"
- "问题是——你打算在原地站多久？"
- "你确定你是在思考，还是在逃避？"
- "如果连试都不敢试，你凭什么说自己尽力了？"

技巧：
- 反问不需要回答，答案已经在问题里
- 一篇文章最多用 2 个反问，多了就成说教
- 最强的反问 = 读者心里早就有答案但不敢承认的那个问题
```

### 1.4 口语化短句

不是网络用语，是真正说话的方式。像朋友深夜跟你掏心窝子。

```
示例：
- "说真的，我也不知道答案。"
- "后来想想，也没那么难。"
- "算了，不骗你了。"
- "就这么简单。但你就是做不到。"
- "对，就是你想的那样。"

技巧：
- 穿插在正式论述中，打破节奏
- 一篇 2000 字的文章用 3-5 句就够
- 位置：段落转折处、承认脆弱处、揭示真相前
- 不要每段都口语化（那就变成没有深度的闲聊了）
```

## 二、中文金句公式

### 2.1 "不是 A，是 B"式

Dan Koe 的 "X is not Y, it's Z" 在中文里的原生表达。

```
英文思维：Productivity is not doing more. It's doing what matters.
翻译体：  生产力不是做更多。而是做重要的事。
中文原生：忙，不叫努力。忙对了，才叫。

英文思维：You're not lazy. You're afraid.
翻译体：  你不是懒惰。你是害怕。
中文原生：你哪是懒，你是怕做了也没用。

更多示例：
- "拖延不是病，是你身体在替你说'不'"
- "焦虑不是问题，焦虑是你还在乎的证据"
- "不是选择太多，是你不敢放弃"
```

### 2.2 场景一击式

一个具体场景，不解释，读者自己悟。

```
- "凌晨三点改完方案，发到群里，老板回了个'嗯'。"
- "收藏夹里 200 篇'改变人生的方法'，人生没变。"
- "买了三本时间管理的书，放在书架上吃了一年灰。"
- "打开朋友圈编辑了五分钟，然后点了取消。"
- "简历改到第七版，投出去的还是第三版。"

技巧：
- 不需要点评，场景本身就是观点
- 越具体越有力（具体到时间、数字、动作）
- 读者看到会想"这不就是我吗"
- 适合做故事积木和反常识钩子的素材
```

### 2.3 转折金句

前半句顺着读者的认知走，后半句突然转弯。

```
- "道理我都懂，但手不听脑子的。"
- "计划很完美，执行很感人。"
- "理论上你已经成功了，实际上你还在刷手机。"
- "你说你想改变，但你连闹钟都不肯早调五分钟。"
- "所有人都说你很优秀，只有你知道自己多害怕。"

技巧：
- 前半句让读者点头
- 后半句让读者扎心
- 转折越日常、越具体，杀伤力越大
```

### 2.4 留白式

说一半，留一半。让读者自己填上最狠的那句。

```
- "后来呢？后来就没有后来了。"
- "你知道最可怕的是什么吗？不是失败。是……"
- "他说'我准备好了'。其实他等的不是准备好，是——"
- "二十岁觉得来得及。三十岁觉得来得及。然后……"

技巧：
- 省略号和破折号是武器，但一篇文章最多用 2 次
- 留白的前提是前文已经建立了足够的张力
- 如果前文没铺垫，留白就只是故弄玄虚
```

## 三、平台语感对照

同一个观点，不同平台的表达方式完全不同。

**核心观点**：拖延的本质不是懒，而是恐惧。

### 小红书版

```
姐妹们，拖延真的不是因为懒❗

你有没有发现，越重要的事越拖？
不是你不想做，是你的大脑在"保护"你 👇

它的逻辑是：
做了 → 可能失败 → 失败 = 被否定 → 太痛了 → 别做了

所以你不是懒，你是怕。

下次拖延的时候问自己：
「我到底在怕什么？」
这一个问题，比任何时间管理 app 都管用 💡
```

### 公众号版

```
你有没有注意到一个反直觉的现象——

越重要的任务，你越拖。
不重要的琐事，反而随手就做了。

如果拖延是因为懒，你应该什么都拖才对。但你没有。你只拖那些"做了可能会失败"的事。

这说明拖延不是能力问题，是保护机制。

你的大脑在做一道简单的风险计算：
做 → 可能失败 → 失败意味着被评判 → 评判带来痛苦 → 不做 = 安全。

所以每次你"拖延"，其实是大脑在说：这件事的潜在痛苦超过了不做的代价。

真正要解决的不是"如何逼自己开始"，
而是"我在害怕什么评判"。
```

### 知乎版

```
先说结论：拖延的本质不是意志力缺失，而是恐惧驱动的回避行为。

证据很简单——如果拖延 = 懒，你应该对所有任务一视同仁地拖。但事实是，你只拖"高风险"任务。给朋友回消息不拖，改老板的 PPT 拖三天。

心理学上这叫"自我妨碍"（self-handicapping）。Berglas & Jones 1978 年的经典实验就证明了：当人们预期自己可能失败时，会主动创造障碍——这样失败了可以归咎于障碍，而非能力。

拖延就是你给自己设的障碍。

"我不是做不好，我是没时间做。"——这句话听着耳熟吗？

所以解法不在时间管理，在于降低"失败 = 被否定"这个等式的权重。具体怎么做，展开说说……
```

### Twitter/X 版

```
拖延不是懒。

拖延是你的大脑在说：
"做了可能失败，失败了会被评判，太痛了——算了别做了。"

你不是缺少自律。
你是缺少安全感。

下次拖延时别问"怎么开始"。
问"我在怕什么"。
```

## 四、翻译体 → 中文原生 速查对照表

| 翻译体 | 中文原生 |
|--------|---------|
| 你不在你想去的地方，因为你还不是会在那里的人 | 不是路太远，是你还没变成走得到的那个人 |
| 目标不是目的地，而是感知的镜头 | 目标不是终点，是滤镜——你盯着什么，就只看得见什么 |
| 生产力不是做更多事，而是做对的事 | 忙是最体面的偷懒 |
| 你的独特性是你最后的护城河 | 能被复制的，迟早被替代 |
| 大多数人试图跳过身份直接改变行为 | 你想着"我要早起"，但你骨子里还是个熬夜的人 |
| 好奇心是最被低估的生产力工具 | 感兴趣的事不用人催，不感兴趣的事闹钟也没用 |
| 多兴趣不是劣势，是现代通才的基石 | 什么都想学不是散，是你还没找到把它们串起来的那根线 |
| 价值型内容已死，视角型内容崛起 | "五个方法"谁都能写，"为什么你不需要方法"只有你能写 |
| 恐惧不是障碍，是信号 | 怕，说明你在乎。不怕的事，不值得做 |
| 在不确定性中找到确定性 | 没有人知道路对不对，但站着不动一定不对 |

**使用规则**：
- 生成金句时，先写中文原生版，不要从英文翻译
- 如果发现自己在翻译 Dan Koe 的英文句式 → 停下来，用本表中的方式重新造句
- 右列不是固定答案，是语感参考。每次应该根据具体主题重新创作

## 五、中文特有的修辞武器

### 5.1 数字具象化

中文读者对具体数字极度敏感。

```
❌ "很多人都有这个问题"
✅ "我咨询过的 200 个客户里，187 个都在第三步放弃了"

❌ "坚持很重要"
✅ "21 天养成习惯是假的。伦敦大学的研究说，平均需要 66 天"

❌ "浪费了很多时间"
✅ "算了一下，过去三个月我花了 147 个小时在'准备开始'上。真正做的时间？11 个小时"
```

### 5.2 时间锚点

用具体的时间点代替模糊的时间描述。

```
❌ "很久以前我也这样"
✅ "2019 年冬天，我还在出租屋里对着空白文档发呆"

❌ "后来我改变了"
✅ "转变发生在去年 3 月的一个周三下午——老板第四次打回我的方案"

❌ "我花了很长时间才明白"
✅ "我花了 14 个月、3 次失败和 1 次差点放弃，才真正理解这句话"
```

### 5.3 身体感受描写

中文叙事中，身体反应比心理描述更有感染力。

```
❌ "我当时非常紧张"
✅ "手心全是汗，心跳快到能听见自己太阳穴在跳"

❌ "她很失望"
✅ "她没说话，只是把手机屏幕朝下扣在桌上"

❌ "我决定做出改变"
✅ "那天晚上我失眠到凌晨四点，第二天顶着黑眼圈交了辞职信"

技巧：
- 用动作代替情绪词
- 用身体反应代替心理描述
- 读者会通过动作自己"感受到"情绪，比你直接说更有力
```

## 六、禁用词/句清单

以下表达已被 AI 用滥，在任何平台都显得机械。看到就换。

| 禁用 | 为什么 | 替代思路 |
|------|--------|---------|
| "干货" | 被用烂了，且暗示内容密度高但往往注水 | 直接证明内容有用，不用贴标签 |
| "赋能" | 企业黑话，普通人不这么说话 | 说清楚具体帮了什么 |
| "颗粒度" | 同上 | "细节""具体程度" |
| "底层逻辑" | 被滥用到失去意义 | 直接说"原因""机制""本质" |
| "认知升级" | 太虚 | 说清楚升级了什么认知，用例子 |
| "破局" | 爽文词汇 | 说清楚怎么解决的 |
| "闭环" | 企业黑话 | "形成循环""前后呼应" |
| "抓手" | 同上 | "入手点""具体方法" |
| "对齐" | 同上 | "达成共识""同步" |
| "拉齐" | 同上 | "统一""拉平" |
| "打法" | 同上 | "方法""策略" |
| "心智" | 常被滥用 | 用在方法论术语中可以，正文叙述中用"想法""思维" |
